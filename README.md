# RNViewer - RN知识库浏览工具

> RNViewer 是一个基于 Streamlit 构建的 Reminds Notes 笔记浏览工具，提供笔记搜索、标签浏览、Markdown 渲染、AI 内容提炼等功能。

**开发者**: De-hamster

---

## 功能特性

- 🔍 **笔记搜索** - 全文搜索笔记标题和内容
- 📁 **标签浏览** - 按标签分类查看笔记
- 📝 **Markdown 渲染** - 支持图片、链接、代码高亮
- 📊 **统计信息** - 查看笔记总数、标签数量等
- 💾 **导出功能** - 导出单篇笔记或整标签笔记
- ✨ **AI 内容提炼** - 使用大模型智能总结笔记内容

---

## 解决了什么？
- 1.批量导出笔记存在空笔记；

- 2.无法合并为一个文档导出；

- 3.无标签的笔记没有单独显示；

- 4.无法选择大量笔记进行总结提取；

- 5.提取笔记指定标签时必须指定主题；

- 6.非VIP总结笔记次数有限；

- 7.不能接入自己已有的大模型API；

- 8.MCP无法提取超过50条笔记的内容；

## 安装步骤

### 方式一：一键启动（推荐）

1. **安装 Python**
   - 下载地址: https://www.python.org/downloads/
   - 安装时勾选 "Add Python to PATH"
   - 推荐版本: Python 3.10 或 3.11

2. **运行启动脚本**
   ```
   双击运行 start.bat
   ```
   脚本会自动检查并安装依赖，然后启动应用。

3. **访问应用**
   启动后在浏览器访问: http://localhost:8501

### 方式二：手动安装

1. **安装依赖**
   ```bash
   pip install -r requirements.txt
   ```

2. **启动应用**
   ```bash
   python -m streamlit run Rnv.py
   ```

## 配置说明

### 数据库配置

首次使用需要在 **设置 → 数据源** 中配置 Reminds 数据库路径：

- **默认路径**:

  Windows是`C:\Users\<用户名>\AppData\Roaming\reminds\reminds.db`
  
  Mac是在`/Users/<用户名>/Library/Application\ Support/reminds/reminds.db`
- **自定义路径**: 如果数据库在其他位置，请手动填写

### AI 提炼配置

在 **设置 → 模型设置** 中配置大模型 API：

| 配置项 | 说明 |
|--------|------|
| Base URL | API 服务地址，如 `https://api.openai.com/v1` |
| API Key | API 密钥 |
| 模型名称 | 使用的模型，如 `gpt-4o-mini` |
> 作者测试过deepseek和阿里百炼coding plan（兼容openai模式）均正常

## 使用指南

### 浏览笔记

1. 点击侧边栏 **全部笔记** 查看统计信息和所有标签
2. 点击标签卡片进入标签详情页
3. 点击笔记卡片的 **查看** 按钮阅读笔记内容

### 搜索笔记

1. 在首页的搜索框输入关键词
2. 搜索结果会显示匹配的笔记列表

### 导出笔记

- **单篇导出**: 在笔记详情页点击 **导出 MD**
- **批量导出**: 在标签页点击 **导出该标签下的所有笔记**

### AI 内容提炼

1. 点击侧边栏 **内容提炼**
2. 选择要提炼的标签
3. 选择提炼类型（综合总结/提取要点/生成大纲/自定义）
4. 点击 **开始提炼** 等待结果

## 启动与停止

### 启动应用

```
双击运行 start.bat
```

启动脚本会自动：
1. 检查 Python 环境
2. 检查并安装缺失的依赖包
3. 检查端口 8501 是否被占用，若被占用则自动释放
4. 启动应用

### 停止应用

- **方式一**: 双击运行 `stop.bat`
- **方式二**: 在启动窗口按 `Ctrl + C`
- **方式三**: 直接关闭命令行窗口

## 常见问题

### Q: 启动后浏览器没有自动打开？

A: 查看命令行窗口显示的地址，手动在浏览器中打开（通常是 http://localhost:8501）

### Q: 图片无法显示？

A: 笔记的图片是安装环境的相对位置，需要在代码中转换，如果图片太多，建议git代码让AI解析做转换。


### Q: AI 提炼功能报错？

A: 检查 API 配置是否正确，确保网络可以访问 API 服务
